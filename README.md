# Forme - 글래스모피즘 운동 앱

모바일 최적화된 글래스모피즘 디자인의 운동 앱입니다.

## 주요 기능

- **실시간 퍼포먼스 수치**: 칼로리, 심박수, 거리 실시간 표시
- **구간별 강도 그래프**: 시간대별 운동 강도 시각화
- **오디오 컨트롤러**: 음악 재생/일시정지, 볼륨 조절, 곡 변경
- **운동 제어**: 일시정지 및 긴급 중단 버튼
- **하단 애니메이션**: 파도 효과와 플로팅 요소

## 디자인 특징

- **글래스모피즘**: 반투명 유리 효과의 모던한 UI
- **배경색**: #F0EEE9 (부드러운 베이지 톤)
- **포인트 컬러**: #FF8A6B (코랄 느낌의 낮은 채도 오렌지)
- **반응형 디자인**: 모바일 화면에 최적화

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
